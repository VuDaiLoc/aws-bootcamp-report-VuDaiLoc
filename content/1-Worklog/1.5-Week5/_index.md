### Week 5 Objectives:

* Understand an overview of Compute services on AWS and the role of Amazon EC2.
* Know how to choose Instance Types, and manage AMIs, Key Pairs and EBS Volumes appropriately.

### Results achieved in week 5:

* Clearly understood the role of Amazon EC2 in AWS Compute infrastructure.
* Learned how to choose the appropriate Instance Type for each workload type (CPU-bound, memory-bound, ...).
* Successfully created and used an AMI to back up and duplicate an EC2 Instance.
* Configured a Key Pair and connected via SSH to an EC2 Instance securely and successfully.
* Successfully created, attached and managed an EBS Volume; set up snapshots for periodic data backup.
