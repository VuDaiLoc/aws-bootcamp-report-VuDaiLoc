### Week 7 Objectives:

* Understand the AWS Shared Responsibility Model.
* Master Amazon IAM, Amazon Cognito, AWS Organizations and AWS Identity Center.

### Tasks to be carried out this week:
| Day | Task | Start Date | Completion Date | Reference Material |
| --- | --- | --- | --- | --- |

| 3 | - Learn about Amazon IAM: managing users, groups, roles and policies; distinguishing Identity-based policy from Resource-based policy; practice assigning permissions following the Least Privilege principle. | 02/06/2026 | 02/06/2026 | <https://000002.awsstudygroup.com> |

### Results achieved in week 7:

* Clearly understood the Shared Responsibility Model, distinguishing security "of the cloud" (AWS) from "in the cloud" (customer).
* Mastered managing IAM Users/Groups/Roles/Policies and proficiently applied the Least Privilege principle.
* Understood and configured Amazon Cognito User Pool/Identity Pool to authenticate users for an application.
* Learned how to organize multiple AWS accounts using AWS Organizations and control them with Service Control Policies.
* Understood the benefits of AWS Identity Center for centralized access management and Single Sign-On (SSO).
