### Week 6 Objectives:

* Understand an overview of AWS storage services.
* Know how to configure security and deploy a static website on Amazon S3.

### Tasks to be carried out this week:
| Day | Task | Start Date | Completion Date | Reference Material |
| --- | --- | --- | --- | --- |

### Results achieved in week 6:

* Grasped the main groups of AWS storage services: Object Storage (S3), Block Storage (EBS), File Storage (EFS/FSx).
* Clearly understood bucket, object and storage class concepts, and how to set up a Lifecycle Policy to optimize storage costs.
* Successfully configured ACL and Bucket Policy, applying the Least Privilege principle when granting S3 access.
* Successfully deployed a static website on Amazon S3, configuring CORS to allow cross-domain resource access.
* Understood the role of the AWS Snow Family and Storage Gateway in large-scale data migration/backup.
